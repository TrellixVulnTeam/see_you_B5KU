# See You

<br />
.NET 6 | Angular based social media prototype.
<br />
<br />
<br />

### To Run This Project 
___
<br />

>API 
> 
>```dotnetcli
>dotnet watch run
>```   
>[runs on port:7027 | **https://localhost:7027**]
>
><br />
>Client 
> 
>```bash
>ng serve
>```
>[runs on port:4200 | **https://localhost:4200**]
